# Learn-hiragana.

ru 

Изучение хираганы (Hiragana Trainer)Простой тренажер для проверки знаний японской азбуки хирагана. Программа показывает случайный японский символ, а вам нужно ввести его правильное звучание на английском языке (ромадзи). В конце тренировки вы увидите свой итоговый результат.
✨ ОсобенностиВыбор языка: Перед началом тренировки можно настроить язык интерфейса программы.Ввод на английском: Проверка ответов происходит строго по системе ромадзи (например: ka, shi, tsu).Статистика: В конце раунда выводится точное количество правильных ответов.
🚀 Как запустить
Требования:
Установленный Python 3.x
🚀Запуск
Скачайте файл japtren.py на рабочий стол!
Откройте терминал и введите: cd Desktop
Запустите программу следующей командой:
japtren.py

eng

Hiragana Trainer: A simple tool for testing your knowledge of the Japanese hiragana script. The program displays a random Japanese character, and you must enter its correct pronunciation in English (romaji). Your final score is displayed at the end of the session.
✨ Features: Language selection: You can configure the program's interface language before starting. Answer checking: Answers are verified strictly using the romaji system (e.g., ka, shi, tsu). Statistics: The exact number of correct answers is displayed at the end of the round.
🚀 How to run
Requirements:
Python 3.x installed
🚀 Launch
Download the `japtren.py` file to your desktop!
Open your terminal and type: `cd Desktop`
Run the program with a single command:
`japtren.py`
